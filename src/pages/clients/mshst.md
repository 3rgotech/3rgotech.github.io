Développeur Web Fullstack pour le compte de la [Maison des Sciences de l'Homme et de la Société de Toulouse](https://mshs.univ-toulouse.fr/), Unité de Service et de Recherche rattachée à l'Université Fédérale de Toulouse et au CNRS.

Réalisation de 3 projets Open-Source au forfait.

# AGAPE

Application web de gestion des Appels à Projets de Recherche, permettant aux utilisateurs inscrits de candidater sur les Appels à Projets de la MSHS-T en saisissant les informations nécessaires à l'évaluation. Cette dernière est effectuée sur l'application par des experts désignés par le comité d'attribution.

Durée du projet : 16 jours/homme
Technologies : PHP 7.3, Laravel 5.7, MariaDB 10.1, Bootstrap 4
Code source : [Github](https://github.com/MSHS-T/agape)

# ERC-MAP

Application de saisie et consultation des données du [Projet de Recherche Européen ERC MAP (Mapping Ancient Polytheisms)](https://map-polytheisms.huma-num.fr/a-propos/), porté par le [Laboratoire PLH (Patrimoine Littérature Histoire)](https://plh.univ-tlse2.fr/). Le projet a pour but d'analyser et cartographier les systèmes religieux de l'Antiquité.

Durée du projet : 45 jours/homme
Technologies : PHP 7.3, Symfony 4.2, PostgreSQL 9.6, PostGIS 2.4, Bootstrap 4
Code source : [Github](https://github.com/LaboratoirePLH/ERC-MAP)

# Expériences CCU

Application de gestion pour la [Plateforme CCU (Cognition, Comportement et Usages)](https://octogone.univ-tlse2.fr/accueil/navigation/plateformes-br-plateaux-techniques/plateforme-ccu-cognition-comportement-et-usages/), dont l'objectif est l'acquisition de données comportementales et physiologiques dans le cadre d'expérimentations. L'application permet à l'équipe de chercheurs de gérer leurs salles et leur matériel, et aux volontaires de s'inscrire sur l'expérimentation et le créneau de leur choix.

Durée prévisionnelle du projet : ~25 jours/homme
Technologies : PHP 7.3, Laravel 6, MariaDB 10.1, ReactJS 16, MaterialUI 4.5
Code source : [Github](https://github.com/MSHS-T/experiences-ccu/)
